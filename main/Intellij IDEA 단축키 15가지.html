Intellij IDEA 단축키 15가지


Alt + Enter

1. Alt+ Enter 키보드 단축키를 사용하면 무엇이든 수정할 수 있습니다
2. 오류가 발견된 경우 해당 오류 위에 커서를 놓고 Alt+ Enter를 누르면 문제에 대한 수정 제안 목록이 표시됩니다
3. 경고 및 제안이 표시되는 곳에서도 사용 가능합니다
4. 오류, 경고 제안이 없는 코드에서도 활용하여 보다 더 간결한 코드를 만들 수 있습니다
5. 경고 및 제안이 표시되는 곳에서도 사용 가능합니다

F2

1. F2를 활용해 손쉽게 다음 오류, 경고, 제안으로 이동할 수 있습니다

Cmd+1(mac), Alt+1(win)

1. 프로젝트 창을 열 수 있습니다
2. 방향키를 사용해 트리를 탐색하고, 타이핑을 하여 검색할 수 있습니다
3. Esc를 누르면 에디터로 돌아올 수 있습니다

ESC

1. 변경 사항을 적용하지 않고 모든 팝업을 닫습니다

Cmd+E(mac), Ctrl+E(win)

1. 최근 파일 대화상자가 열려 방향키로 파일을 탐색할 수 있습니다
2. 최근 파일 대화상자에서 도구 창을 열 수도 있으며 키워드 입력을 통해 특정 항목을 검색할 수 있습니다

Cmd+B(mac), Ctrl+B(win)

1. 심볼의 선언으로 이동할 수 있습니다
2. Alt를 같이 누르면 구현을 탐색하는 것도 가능합니다

Alt + F7

1. 해당 변수명의 모든 사용 위치가 검색창에 표시됩니다

Tap Ctrl Twice

1. IDE에서 어떤 위치에 있든, 어떤 파일이 열려 있든 Ctrl을 두 번 누르면 Run Anything 창이 열립니다
2. Run Anything 창은 최근 실행한 실행 구성 목록이 표시되고 실행할 항목의 이름을 입력하여 다른 실행 구성을 검색할 수 있습니다

Alt+ 위/아래 방향키(mac), Ctrl+W, Ctrl+Shift+W(win)

1. 커서 주변의 코드 섹션을 확장하거나 축소할 수 있습니다
2. 선택 영역 확장을 사용하여 확장된 섹션의 유효한 표현식을 자동으로 선택합니다
3. 유효한 Java 블록을 통해 선택 영역이 축소되어 커서로 돌아옵니다

Cmd+/(mac), Ctrl+/(win)

1. 줄 어디서든 코드를 간편하게 줄 주석으로 처리됩니다
2. 이미 주석 처리된 줄에서 같은 단축키를 다시 누르면 주석 처리가 해제됩니다
3. 전체 블록을 선택한 경우 Alt+Cmd+/ 또는 Shift+Ctrl+/를 눌러 블록 주석을 추가할 수 있습니다

Shift+Cmd+Enter(mac), Shift+Ctrl+Enter(win)

1. 세미콜론을 추가하거나 for문, if문에 구문을 완성시킵니다
2. 커서가 그 다음 필요한 위치로 올바르게 배치됩니다

Alt+Cmd+L(mac), Ctrl+Alt+L(win)

1.현재 파일의 서식을 프로젝트 표준에 맞출 수 있습니다
2. Alt+Shift+Cmd+L(mac), Shift+Ctrl+Alt+L(win)을 눌러 옵션을 선택해 서식을 지정할 수 있습니다

Ctrl+T(mac), Shift+Ctrl+Alt+T(win)

1.심볼이나 선택 영역에서 적용 가능한 모든 리팩토링을 표시할 수 있습니다
2. 리팩토링 이름 옆의 숫자를 사용하여 원하는 리팩토링을 선택할 수도 있습니다
3. 특정 리팩도링의 키보드 단축키가 존재할 경우 해당 단축키가 표시됩니다

Shift+Cmd+A(mac), Shift+Ctrl+A(win)

1. IntelliJ IDEA의 모든 액션을 검색할 수 있습니다
2. 액션찾기 창에는 액션뿐만 아니라 단축키도 함께 표시되므로 단축키를 익히고 연습할 수도 있습니다
3. 설정도 검색 가능해서 액션찾기 창에서 설정을 바로 변경할 수 있습니다
4. 도구 창 검색 및 열기도 지원됩니다

Tap Shift Twice

1. Shift를 두 번 누르면 무엇이든 검색 가능한 검색 창이 표시됩니다
2. 액션 검색과 마찬가지로 검색 창에서 설정을 변경할 수 있고 명령어도 지원합니다
3. 기본적으로 최근 파일이 표시되므로 Cmd+E 대신 이 창을 활용할 수 있습니다
4. 검색어를 입력하면 클래스, 파일, 심볼 및 액션의 검색 결과가 모두 표시됩니다
